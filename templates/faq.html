FAQ

  • What is going on with my data ?

Your data is uploaded to the server where it is processed to extract the information necessary to display the analysis. When the your chat file is loaded in the server, it is instantly deleted. No information is kept on the server.

  • What data is stored ?

The chat file is temporally stored in the server to be loaded in a pandas dataframe. When the data processing begins it is deleted for the server. If an error is met while processing your data it is still deleted. Hence, none of your data is stored.

  • Are my data and analysis discoverable ?

In order to keep your chat analysis, it is only available on the file upload. The analysis of your chat is not stored and is unavailable as soon as you leave the page.

  • Are cookies collected ?

The only cookies used on this website are the ones from the google analytics service. See more information.
